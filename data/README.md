# BioTriplex Data Files

This directory contains the data files for the BioTriplex dataset, which is used for benchmarking biomedical Relation 
Extraction and Named Entity Recognition models. The dataset includes annotated text files with biomedical entities and 
their relationships.

## Directory Structure
- `Annotated Full Text Paper Folders/`: Contains subfolders for each paper with annotated `.xml` files with entity and 
  relation annotations for different sections of PubMedCentral full-text articles.
- `preprocessed_dataset/`: Contains preprocessed data files ready for use with various models.
